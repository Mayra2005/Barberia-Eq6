DESCRIPCION:
El proyecto consiste en el desarrollo de un Sistema de Gestión de Citas y Administración para la Barbería “Super Bee”, cuyo propósito es digitalizar y optimizar los procesos administrativos y operativos del negocio. 

Actualmente, la barbería lleva el control de citas, ingresos, egresos y servicios realizados de manera manual en un cuaderno, lo que puede generar errores, pérdida de información y dificultades en el control administrativo. 

El sistema propuesto permitirá automatizar la gestión de citas, empleados, promociones, estilos, pagos y respaldo de información, facilitando la organización interna y mejorando la eficiencia del servicio. 

El sistema será de uso interno y únicamente interactuarán con él el dueño y los barberos, cada uno con permisos específicos según su rol. Esto permitirá un mejor control administrativo, reducción de errores humanos y una administración más moderna y eficiente del negocio.

- Desarrollo web: HTML, JavaScript y css
