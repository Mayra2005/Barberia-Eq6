---Uso de Inteligencia Artificial ---

Este documento describe el uso de herramientas de Inteligencia Artificial durante el desarrollo del sistema de gestión para una barbería, enfocándose en la creación del frontend utilizando HTML, CSS y JavaScript.
La IA fue utilizada como una herramienta de apoyo para mejorar la calidad del diseño, optimizar el código y facilitar la implementación de funcionalidades del sistema. Hicimos uso de ChatGPT (OpenAI)


--- Propósito del uso de IA ---

La utilizamos con los siguientes objetivos:
 Apoyar en la creación de interfaces web
 Generar estructuras base de páginas HTML
 Diseñar estilos visuales con CSS
 Implementar funcionalidades con JavaScript
 Resolver errores durante el desarrollo
 Mejorar la experiencia de usuario (UI)

--- Se utilizó IA para: ---

Crear la estructura base de las páginas:
 Inicio de sesión
 Registro de usuarios
 Agendar citas
 Gestión de empleados
 Gestión de pagos, etc
 Generar formularios con:
 Campos de entrada (inputs)
 Botones
 Validaciones básicas
 Organización del contenido mediante etiquetas semánticas

---Diseño y estilos---
La IA ayudó en:
 Aplicación de estilos visuales modernos
 Selección de colores adecuados para la barbería
 Alineación y distribución de elementos (layouts)
 Uso de clases y selectores CSS
 Diseño responsive básico para diferentes pantallas
 Mejora de la apariencia de formularios y botones

---Funcionalidad---
Se utilizó IA para:
 Implementar validaciones de formularios
 Manejo de eventos (clicks, envíos de formularios)
 Interacción dinámica con el usuario
 Simulación de funcionalidades del sistema (ej. registro o citas)
 Corrección de errores en scripts

--- Uso de IA en la documentación ---
La IA también apoyó en la organización de la documentación del proyecto.

--- Consideraciones éticas ---
El uso de la inteligencia artificial se realizó bajo los siguientes principios:
 Uso responsable de la tecnología
 Comprensión del código generado
 Validación de resultados obtenidos
 No dependencia total de la IA
 Respeto a buenas prácticas de desarrollo

--- Beneficios obtenidos ---
El uso de IA permitió:
 Reducir el tiempo de desarrollo
 Mejorar el diseño visual del sistema
 Facilitar el aprendizaje de HTML, CSS y JavaScript
 Detectar y corregir errores rápidamente
 Mejorar la organización del frontend

La IA fue utilizada únicamente como apoyo, no como reemplazo del trabajo del equipo.
